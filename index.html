Animaciones en css

Que son las animaciones

Las animaciones en CSS son técnicas que permiten cambiar de forma gradual el estilo de un elemento HTML sin necesidad de usar JavaScript. Estas animaciones pueden modificar propiedades como colores, tamaños, posiciones, rotaciones, opacidad y más.

Transiciones (transition): cambian suavemente de un estado a otro.
Animaciones (animation + @keyframes): permiten movimientos y cambios más complejos y automáticos.

Ventajas y usos principales

Ventajas
  • Mejoran la apariencia visual de una página web.
  • Hacen más interactiva la experiencia del usuario.
  • Ayudan a destacar elementos importantes.
  • Son más ligeras y rápidas que muchas animaciones hechas con JavaScript.
  • Funcionan en la mayoría de navegadores modernos.
USos principales
  • Efectos al pasar el mouse sobre botones.
  • Menús interactivos.
  • Cargas y pantallas de espera.
  • Movimiento de imágenes o textos.
  • Aparición y desaparición de elementos.

Transiciones en CSS

Propiedad transition

La propiedad transition permite que un cambio de estilo ocurra de manera suave durante un tiempo determinado.

transition-property

Define qué propiedad tendrá la transición.

transition-property: background-color;

transition-duration

Indica cuánto tiempo durará la transición.

transition-duration: 3s;

transition-timing-function

Controla la velocidad de la animación.

transition-timing-function: ease;

transition-delay

Define cuánto tiempo esperar antes de iniciar la transición.

transition-delay: 1s;

Transformaciones en CSS

Propiedad transform

La propiedad transform permite modificar la forma, tamaño o posición de un elemento.

translate()

Mueve un elemento.

transform: translate(50px, 20px);

rotate()

Rota un elemento.

transform: rotate(45deg);

scale()

Aumenta o reduce el tamaño.

transform: scale(1.5);

skew()

Inclina un elemento.

transform: skew(20deg);

Animaciones en CSS

Propiedad animation

La propiedad animation en CSS es una forma abreviada (shorthand) para configurar animaciones en elementos HTML, permitiendo definir el movimiento, cambios de estilo y secuencias complejas sin usar JavaScript.

Uso de @keyframes

La regla @keyframes en CSS es una herramienta fundamental que permite crear animaciones personalizadas definiendo estilos CSS en puntos específicos (fotogramas clave) a lo largo de una secuencia. Permite cambiar gradualmente el estilo de un elemento desde un conjunto de CSS a otro, creando transiciones fluidas y complejas, más allá de lo que permiten las transiciones simples.

Parámetros principales

animation-name

Nombre de la animación.

animation-name: mover;

animation-duration

Tiempo que dura la animación.

animation-duration: 3s;

animation-iteration-count

Número de repeticiones.

animation-iteration-count: infinite;

animation-delay

Tiempo antes de iniciar.

animation-delay: 2s;

Diferencias entre transiciones y animaciones

Cuándo utilizar cada una

Usar transiciones cuando:

  • Se necesite un efecto simple.
  • El cambio dependa del usuario.
  • Se quieran efectos suaves en botones o enlaces.

Usar animaciones cuando:

  • Existan varios pasos o movimientos.
  • El efecto sea automático.
  • Se requieran repeticiones continuas.

Ventajas y desventajas

Transiciones

Ventajas

  • Fáciles de usar.
  • Código corto.
  • Buen rendimiento.

Desventajas

  • Limitadas a cambios simples.
  • Necesitan interacción o cambio de estado.

Animaciones

Ventajas

  • Más control y personalización.
  • Permiten secuencias complejas.
  • Pueden repetirse automáticamente.

Desventajas

  • Código más extenso.
  • Pueden afectar el rendimiento si son excesivas.
